# JADLE
An API app that returns restaurant data.
## Setup/Installation Requirements
* Fork/Clone the repository
```
   $ git clone https://github.com/andreassenmarvin/Jadle.git
```
* Open your project on IntelliJ
* Run project
```
   $ gradle run
```
